# Cloud Providers & CI/CD Tools

This guide covers major cloud providers and CI/CD tools essential for modern DevOps practices.

## Cloud Providers

### Amazon Web Services (AWS)

- **Core Services:**
  - EC2: Virtual servers
  - S3: Object storage
  - RDS: Managed databases
  - Lambda: Serverless computing

### Microsoft Azure

- **Key Features:**
  - Virtual Machines
  - App Services
  - Azure Functions
  - Azure DevOps

### Google Cloud Platform (GCP)

- **Popular Services:**
  - Compute Engine
  - Cloud Storage
  - Cloud Functions
  - Kubernetes Engine
